# 需求文档：背景音乐系统

> 创建时间: 2026-03-17
> 状态: ✅ 已完成

## 背景

道诡修仙录已实现完整的音效系统（攻击、受击、治愈、胜利、失败等），但缺少背景音乐。背景音乐是游戏沉浸感的重要组成部分，能够：
- 强化场景氛围（主菜单的神秘感、战斗的紧张感、Boss的压迫感、篝火的宁静感）
- 增强游戏节奏感
- 提升整体游戏体验

## 目标

为游戏四个主要场景添加程序化背景音乐：
1. **主菜单** - 神秘、悠远的修仙氛围
2. **普通战斗** - 紧张、有节奏感
3. **Boss战** - 压迫感强、节奏更快
4. **篝火/休息** - 宁静、恢复、治愈感

## 范围

### 包含
- ✅ 扩展 soundManager.ts 添加 BGM 功能
- ✅ 使用 Web Audio API 程序化生成背景音乐（不增加包体积）
- ✅ 实现场景切换时的音乐过渡（淡入淡出）
- ✅ 音量控制（独立于音效音量）
- ✅ 设置持久化（localStorage）

### 不包含
- 外部音频文件加载
- 复杂的音乐旋律生成（保持简单循环）
- 多曲目切换系统

## 验收标准

1. ✅ 主菜单启动时自动播放背景音乐
2. ✅ 进入战斗时切换到战斗BGM
3. ✅ Boss战时切换到Boss BGM
4. ✅ 篝火场景播放休息BGM（商店/事件也使用此BGM）
5. ✅ 返回主菜单时正确切换回主菜单BGM
6. ✅ BGM可以独立于音效开关/调节音量
7. ✅ 设置持久化到localStorage
8. ✅ `tsc --noEmit && vite build` 验证通过
9. ✅ 游戏运行正常，无报错
