# 角色解锁系统

## 背景
代码中已存在 `unlockedCharacters` 字段（默认只有心素），成就系统也有角色通关成就，但角色选择界面所有角色都可选，解锁逻辑未实现。

## 目标
1. 实现角色解锁机制，增加游戏长期目标
2. 新玩家先熟悉心素角色，再解锁其他角色
3. 提供清晰的解锁条件和解锁提示

## 范围
- 角色选择界面显示锁定/解锁状态
- 锁定角色的解锁条件提示
- 成就触发自动解锁角色
- 解锁成功弹窗提示

## 解锁条件设计

### 心素（默认解锁）
- 新玩家默认可用
- 均衡型角色，适合新手

### 法教角色解锁
- **条件**：使用心素通关一次
- **触发**：`first_victory` 成就解锁时
- **设计意图**：让玩家先熟悉游戏基础，再体验其他风格

### 坐忘道角色解锁
- **条件**：在入魔状态下击败Boss
- **触发**：`madness_victory` 成就解锁时
- **设计意图**：体现"以疯证道"的主题，高风险挑战

## 验收标准
- [x] 新存档只有心素可选
- [x] 锁定角色显示锁定图标和解锁条件
- [x] 使用心素通关后解锁法教
- [x] 入魔状态下击败Boss解锁坐忘道
- [x] 解锁时有弹窗提示
- [x] 已解锁角色跨存档保留
