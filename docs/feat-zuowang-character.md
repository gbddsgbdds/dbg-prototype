## 背景

当前游戏有两个角色（心素/李火旺、法教/张道陵），需要增加第三个角色丰富游戏体验。坐忘道是《道诡异仙》中的神秘门派，以「以假乱真、虚实难辨」著称，非常适合作为第三个角色。

## 目标

设计并实现坐忘道角色，作为第三个可选角色：
- 定义角色属性和初始卡组
- 实现「幻象」机制（打出时有概率双倍效果或无效果）
- 更新角色选择界面图标
- 确保角色有独特的游戏风格

| 名称 | 骗子 | 坐忘道代表人物 |

### 角色定位

| 角色 | 风格 | 核心机制 |
|------|------|---------|
| 心素 | 高风险高回报 | 自残→煞气→入魔爆发 |
| 法教 | 稳定防御 | 法术→护盾→持续成长 |
| 坐忘道 | 赌博诡道 | 幻象→随机效果→运气为王 |

### 核心机制：幻象 (Illusion)

**幻象牌**：打出时掷骰子决定效果
- 50% 概率：正常效果
- 25% 概率：双倍效果（真实幻象）
- 25% 概率：无效果（虚假幻象）

这创造了独特的「赌徒」体验——每次出牌都是一次赌博。

### 技术细节

**幻象效果实现**：
```typescript
// 在 playCard 时检查
if (card.illusion) {
  const roll = Math.random()
  if (roll < 0.25) {
    // 双倍效果
    effectMultiplier = 2
    log('🌟 幻象成真！效果翻倍！')
  } else if (roll < 0.5) {
    // 无效果
    log('💨 幻象消散...无效果')
    return // 或设置效果为0
  }
  // 50% 正常效果，不做额外处理
}
```

### 风险
- 幻象机制可能影响游戏平衡
- 需要确保概率分布正确
- 需要在 UI 中清晰提示幻象结果
