# 更新日志

## v0.3.0 (2026-03-17)

### 新功能
- 🔥 **煞气系统**：攻击伤害随煞气加成 0-100%
- 💀 **入魔机制**：煞气满100后触发入魔，伤害×3，5能量，持续8回合
- 🧠 **理智系统**：煞气30+开始扣理智，理智归零=死亡
- ✨ **道心系统**：同体系牌累计道心，10层触发禁忌连段
- 👁️ **幻觉机制**：煞气60+时20%概率出现幻觉牌

### 卡牌
- 初始牌组：12张（挥刀、防御、狂斩、傩面、血刃等）
- 奖励卡池：7张（顺劈斩、铁壁波、嗑血、捷击等）
- 特殊卡牌：走火入魔、驱魔符、禁术·血肉嫁接

### 敌人
- 普通敌人：颌虫、画皮
- 精英敌人：尸煞
- Boss：丹阳子（75HP，阶段切换）

### UI/UX
- 理智条、煞气条显示
- 入魔视觉特效（红色脉冲）
- 道心层数显示
- 敌人类型标识（精英/Boss）
