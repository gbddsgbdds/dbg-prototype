- **休息音乐**

## v0.5.7 (2026-03-17)

### 🔊 音效系统增强
- **高级音频合成技术**
  - 多层叠加：每个音效由多个音层组成，增加丰富度
  - 包络控制：ADSR包络塑造音色动态
  - 滤波器：低通、高通、带通滤波器增强音质感
  - 噪声生成：白噪声、粉红噪声增加真实感
  - 随机变体：参数微调避免重复感
- **重制的9种音效**
  - attack：剑击打击感（低频冲击 + 高频切削 + 金属噪声）
  - hit：受击反馈（低频冲击 + 痛苦高频 + 冲击波噪声）
  - heal：神圣治愈感（大三和弦 + 高频闪烁）
  - block：金属护甲感（金属撞击 + 高频共鸣 + 金属噪声）
  - victory：凯旋旋律（上行大三和弦 + 高音收尾 + 闪烁效果）
  - defeat：阴郁消沉（下行音阶 + 低频隆隆）
  - card：清脆操作感（基础点击 + 纸质感噪声）
  - click：UI交互（简洁高频点击）
  - madness：扭曲不安（不协和和弦 + 低频震动 + 高频噪声）

### 📝 文档更新
- 新增 `docs/feat-enhanced-sound.md` 音效增强需求文档
