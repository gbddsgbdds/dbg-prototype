# 需求：更新日志面板移动端适配

## 背景
更新日志面板（Changelog）是用户查看游戏版本历史的界面，目前只有桌面端样式，缺少移动端响应式适配。

## 目标
为更新日志面板添加移动端响应式样式，确保在小屏幕设备上正常显示。

## 范围
- 仅针对 `.changelog-panel` 及其子元素添加移动端适配
- 适配断点：400px及以下

## 验收标准
1. 在400px及以下屏幕上，面板宽度适应屏幕
2. 标题、内容字体大小适配
3. 内边距、间距合理
4. 滚动正常，内容不溢出

## 开发计划

### 涉及文件
- `src/App.css` - 添加移动端响应式样式

### 实现步骤
1. 在 `@media (max-width: 400px)` 中添加 `.changelog-panel` 相关样式
2. 优化字体大小、内边距、间距
3. 确保滚动流畅
