# 需求文档：卡牌扩展（达到40+目标）

## 背景
当前游戏有26张卡牌，目标为40+张。需要添加约14张新卡牌以丰富策略选择和游戏深度。

## 目标
1. 卡牌总数达到40张以上
2. 丰富各类型卡牌（攻击/技能/能力）
3. 平衡各稀有度分布
4. 保持与道诡修仙录世界观的契合

## 范围
### 包含
- 新增卡牌定义（src/data/cards.ts）
- 卡牌图标（使用placeholder，待后续美术资源）
- 平衡性考虑

### 不包含
- 新机制实现（使用现有机制）
- 角色专属卡牌（已有3个角色）
- UI改动

## 验收标准
- [ ] 卡牌总数 >= 40张
- [ ] 各类型卡牌数量合理分布
- [ ] `tsc --noEmit && vite build` 通过
- [ ] 游戏可正常运行，新卡牌可正常获取和使用

## 开发计划
### 修改文件
