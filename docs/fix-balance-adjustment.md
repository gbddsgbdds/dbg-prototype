# 数值平衡调整

## 背景
游戏已有3个角色、40+卡牌、15种敌人，但数值平衡存在问题：
- 心素初始卡组防御效率低于法教
- 敌人伤害增长曲线与玩家生存能力不匹配
- 部分卡牌效果强度不合理

## 目标
1. 统一角色初始卡组强度基准
2. 调整敌人伤害曲线，确保各层难度递进合理
3. 优化防御卡牌效率

## 范围
- 调整初始卡牌数值
- 微调敌人HP和伤害值
- 不涉及新增机制

## 验收标准
- [ ] 心素初始卡组与法教/坐忘道强度相当
- [ ] 第1层→第2层→第3层难度递进合理
- [ ] tsc --noEmit && vite build 通过

## 开发计划

### 涉及文件
- `src/data/cards.ts` - 卡牌数值调整
- `src/data/characters.ts` - 角色属性确认

### 实现步骤
1. 分析当前各角色卡组强度
2. 调整心素初始卡牌数值
3. 微调敌人伤害曲线
4. 测试验证
