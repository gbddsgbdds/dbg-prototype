# 需求文档：第2层卡池和敌人

> 创建时间: 2026-03-17 02:57
> 状态: ✅ 已完成

---

## 1. 背景

当前游戏只有第1层的卡池和敌人，玩家在第2-4层仍然使用相同的奖励卡池和敌人，缺乏难度递进和内容深度。

## 2. 目标

实现第2层的专属卡池和敌人，让游戏有更丰富的内容和合理的难度曲线。

## 3. 范围

### 3.1 包含
- 设计并实现第2层奖励卡池（6张新卡牌）✅
- 设计并实现第2层敌人（2个新普通敌人 + 1个新精英）✅
- 修改地图生成逻辑，根据层数分配不同的敌人和卡池 ✅
- 确保数值平衡符合GDD ✅

### 3.2 不包含
- 第3层内容（单独任务）
- Boss设计（已有丹阳子）
- UI改动

## 4. 验收标准

- [x] 第2层战斗节点使用新敌人
- [x] 第2层胜利奖励从新卡池选择
- [x] 第1层和第2层敌人/卡池正确区分
- [x] `tsc --noEmit && vite build` 通过
- [x] 游戏可以正常运行并体验第2层内容

#### 第2层新敌人

**普通敌人1：煞鬼**
- HP: 22
- 意图循环: 攻10 / 攻10 / 防8 / 攻12
- 特点: 高攻低防

**普通敌人2：心魔**
- HP: 18
- 意图循环: 防御6 / 偷理智(8) / 攻8 / 攻8
- 特点: 攻击理智

**精英敌人：血傀儡**
- HP: 55
- 意图循环: 攻12 / 攻12 / buff(力量+3) / 攻16
- 特点: 成长型精英
