## 背景

当前敌人总数为12种，目标是达到15+种。需要添加3个新敌人丰富游戏体验。

## 目标

设计并实现3个新敌人：
1. 鬼面（第1层普通敌人）
2. 阴魂（第2层普通敌人）
3. 血影（第3层精英敌人）

## 范围

### 包含
- 敌人定义（类型、HP、意图、特殊机制）
- 敌人数值平衡
- 敌人池更新

### 不包含
- 新敌人图标图片（使用占位）
- 新战斗机制

## 敌人设计

### 1. 鬼面（第1层普通）

| 属性 | 值 |
|------|------|
| id | ghost_face |
| 名称 | 鬼面 |
| HP | 10 |
| 类型 | normal |
| 特点 | 低HP，防御为主 |

意图循环：
- 防御 8
- 防御 6
- 攻击 5
- 攻击 5

定位：新手友好型敌人，让玩家练习防御机制。

### 2. 阴魂（第2层普通）

| 属性 | 值 |
|------|------|
| id | yin_soul |
| 名称 | 阴魂 |
| HP | 20 |
| 类型 | normal |
| 特点 | 攻击理智 |

意图循环：
- 防御 6
- 理智削debuff（sanDrain 8）
- 攻击 9
- 理智削debuff（sanDrain 10）

定位：第2层理智压力敌人，为第3层做准备。

### 3. 血影（第3层精英）

| 属性 | 值 |
|------|------|
| id | blood_shadow |
| 名称 | 血影 |
| HP | 65 |
| 类型 | elite |
| 特点 | 高伤+自愈 |

意图循环：
- 攻击 14
- 攻击 14
- 防御 12
- 攻击 18
- 恢复 15（自愈）

定位：高难度精英，需要速战速决。
