# DBG 数值设计方法论

## 一、杀戮尖塔/怪物火车的已知经验

### 杀戮尖塔的平衡理念（Mega Crit）

1. **玩家应该感觉输是因为自己的决策**，不是随机
2. **"前期难，后期强"** — 新手期死亡率高，但学习后胜率提升
3. **永远有选择的错觉** — 即使牌运不好，也应该有策略翻盘的可能
4. **每张牌都有用** — 所谓"垃圾牌"是相对的，在特定 build 里可能是核心牌

### 怪物火车的数值哲学

1. **数值可以爆炸** — 给玩家创造"上帝模式"的快感
2. **Combo 是核心** — 多个效果叠加产生远超 1+1>2 的效果
3. **风险回报** — 爆数值的牌通常有代价
4. **不要怕 OP** — 让玩家觉得强是乐趣的一部分

### 杀戮尖塔具体数据参考

**伤害基准：**
- Act1 普通敌人 HP: 10-15
- Act1 Boss HP: 60-80
- 玩家 HP: 80
- 每回合获得 3 能量
- 普通攻击牌伤害: 5-9
- 普通防御牌格挡: 4-6
- 每层战斗约 3-4 回合

**数值公式：**
- 基础伤害 = 卡牌伤害 + 力量加成
- 最终伤害 = (基础伤害) × (1 + 易伤比例)
- 实际伤害 = 最终伤害 - 敌人格挡

## 二、DBG 数值设计的五个层次

### 层次 1：单卡价值（微观）

每张牌的价值 = DPT（每回合伤害输出） 或 DPTB（每能量伤害）

**杀戮尖塔的参考：**
- 1费牌 DPTB 合理范围: 5-8
- 2费牌 DPTB 合理范围: 6-10
- 3费牌 DPTB 合理范围: 8-14（需要有额外效果补偿）

**"上限牌"原则：** 一张牌在理想情况下很强是可以的，但不能每回合无条件打出就赢。需要：
- 条件触发（需要前置牌）
- 资源代价（自残、消耗 HP）
- 组合需求（需要特定 build）

### 层次 2：牌组效率（中观）

**核心指标：**
- 平均 DPT（每回合伤害输出）
- 平均 DPTB（每能量伤害）
- 牌组厚度（总牌数 — 越薄越稳定，越厚越不可控）

**杀戮尖塔参考：**
- 5 张初始牌 → 平均 DPTB ≈ 3-4
- 15 张成长后牌组 → 平均 DPTB ≈ 6-8
- 完整 build → 平均 DPTB ≈ 10-15

**我们的"怪物火车式"调整：**
- 因为允许数值爆炸，上限可以提高
- 基础 DPTB: 4-6
- 入魔状态 DPTB: 15-30（允许很高）

### 层次 3：回合 TTK（战斗节奏）

**TTK = Time To Kill（击杀所需回合数）**

```
TTK = 敌人HP ÷ 玩家平均DPT
```

| TTK | 体验 |
|-----|------|
| ≤2 | 秒杀（太无脑） |
| 3-5 | 爽快（怪物火车目标） |
| 6-8 | 标准（杀戮尖塔目标） |
| ≥10 | 拖沓（不好玩） |

**Boss TTK 设计：**
- Act1 Boss: TTK ≈ 6-8 回合（学习阶段）
- 入魔 build: TTK ≈ 2-3 回合（爽感阶段）

### 层次 4：成长曲线（宏观）

**每层/每境界后的 DPT 增长率：**

| 阶段 | DPT 增长率 | 说明 |
|------|-----------|------|
| 初始 | 基准 100% | 基础值 |
| 第一层奖励 | +30-50% | 选卡+少量删除 |
| 中期 build 成型 | +100-150% | 核心 synergy 触发 |
| 入魔/终极 build | +300-500% | 数值爆炸 |

### 层次 5：风险/回报比

**每张牌的风险评估：**

| 风险类型 | 示例 | 回报倍率 |
|---------|------|---------|
| 无风险 | 打击 8 伤害 | 1× |
| 低风险 | 打击 12 伤害 | 1.5× |
| 中风险 | 自扣 3HP，攻击 15 | 1.8-2× |
| 高风险 | 自扣 8HP，攻击 25 + 煞气 | 2.5-3× |
| 极高风险 | 走火入魔，理智归零 | 3-5×（可能死亡） |

**怪物火车原则：高回报允许 OP，但必须有代价。**

## 三、自残/献祭类卡牌的平衡公式

这类牌是本游戏的核心，需要特别设计：

### 通用公式

```
有效伤害 = 卡牌伤害 × (1 + 煞气加成)
实际代价 = 自扣HP + 理智损失风险

性价比 = 有效伤害 ÷ 实际代价
```

**目标性价比区间：**
- 合理: 2-3
- 很强但合理: 3-4
- 超强: 4-6
- 破坏平衡: >6（需要其他限制）

### 示例计算

```
「血刃」: 消耗 5HP，造成 15 伤害，获得 3 煞气
在 0 煞气时: 性价比 = 15 ÷ 5 = 3.0（合理）
在 60 煞气时(+60%): 性价比 = 15 × 1.6 ÷ 5 = 4.8（很强）
在 90 煞气时(+100%): 性价比 = 15 × 2.0 ÷ 5 = 6.0（爆炸）
在入魔时(×3): 性价比 = 15 × 3.0 ÷ 5 = 9.0（God mode）
```

这就是自残牌滚雪球的原理：**前期性价比合理，后期叠到煞气后性价比爆炸。**

## 四、敌人设计数值框架

### 普通敌人

```
敌人HP = 基础HP × 层数系数
基础HP = 12-18
层数系数 = 1.0 (Layer1) → 1.5 (Layer2) → 2.0 (Layer3)

敌人攻击 = 基础攻击 × (1 + 阶段系数)
基础攻击 = 6-10
```

### 精英敌人

```
HP = 普通敌人 × 1.8-2.5
通常有特殊机制而非高数值
机制 > 数值
```

### Boss

```
HP = 普通敌人 × 5-8
分阶段：每个阶段行为模式不同
阶段转换 HP 阈值：30% / 60%
```

### 敌人意图设计

每回合意图的伤害应该 ≤ 玩家 HP 的 25%（允许被打，但不至于被一回合打死）

```
最大单次伤害 ≤ 玩家HP × 0.25
```

## 五、 combosynergy 的平衡原则

### 协同判断标准

好的 synergy 应满足：

1. **有明确信号** — 玩家能自然发现（不需要查 wiki）
2. **有成长曲线** — 一开始薄弱，越叠越强
3. **不是唯一解** — 同一个角色应该有多种 build 方向

### 道心连段的平衡

```
触发条件：打出 10 张同体系牌（合理？太多了）
调整：
→ 需要 8 张同体系牌
→ 允许跨体系组合（兵家3+傩戏3+任意2=触发）
→ 连段效果有使用限制（每场战斗1次）
```

### 道心层数简化

| 道心 | 需要打出牌数 | 效果 |
|------|------------|------|
| 1 | 2 | 小提示（+1%伤害） |
| 2 | 4 | 抽1张牌 |
| 3 | 6 | 攻击+15% |
| 4 | 8 | 攻击+30%，获得护甲 |
| 5 | 10 | **禁忌连段**（一场战斗1次） |

## 六、数值表模板（第一个角色用）

### 卡牌成本公式

```
1费：基础效果，DPTB 5-8
2费：基础效果增强，DPTB 6-10，或带额外效果
0费：纯辅助/Setup，DPTB 0（但提供价值）
3费：强力效果，DPTB 10-14，或有重大副作用
```

### 伤害基准表

| 阶段 | 1费攻击 | 2费攻击 | 3费攻击 | Boss HP |
|------|---------|---------|---------|---------|
| 第1层(练气) | 6-8 | 10-14 | 16-20 | 50-60 |
| 第2层(筑基) | 10-14 | 16-22 | 24-30 | 80-100 |
| 第3层(金丹) | 16-22 | 24-32 | 36-48 | 120-160 |
| 入魔状态 | ×3 | ×3 | ×3 | 秒杀级 |

### 煞气加成表

| 煞气区间 | 攻击加成 | 副作用 |
|---------|---------|--------|
| 0-29 | +0% | 无 |
| 30-59 | +30% | 每回合-2理智 |
| 60-89 | +60% | 每回合-5理智，出现幻觉 |
| 90-99 | +100% | 每回合-10理智，强制出牌 |
| 100 | +200%（入魔） | 理智归零风险 |

## 七、验证流程

设计完数值后，用以下方式验证：

1. **纸面模拟**：手动走一遍完整战斗，检查 DPT/TTK
2. **极限测试**：最优 build 和最差 build 分别打一遍
3. **"爽感测试"**：最佳 combo 能打出多少伤害？
4. **"受虐测试"**：最差情况玩家能撑几回合？
5. **蒙特卡洛模拟**：跑 1000 次随机模拟，检查胜率分布

## 八、我们的数值策略（刻意不平衡）

既然目标是"放大爽点、弱化不爽"：

1. **不追求完美平衡** — 允许 OP build 存在
2. **自残牌的性价比要够高** — 让玩家觉得"流血换力量是值得的"
3. **入魔状态要够爽** — 但有死亡风险作为制衡
4. **Boss 不用太难** — 玩家应该主要死于"叠煞气太猛自杀"，不是死于Boss打人
5. **幻觉机制适度** — 增加趣味但不破坏策略性

核心矛盾：**玩家越强越容易死**（因为自残太多），这本身就是游戏乐趣的一部分。
