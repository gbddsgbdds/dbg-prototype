# 第4层专属内容扩展

## 背景
游戏第4层是最终Boss层，但目前内容与第3层完全重复（敌人、卡池、精英都是复用的）。这让最终挑战缺乏独特性和层次感，玩家在第4层的体验与第3层几乎一致。

## 目标
1. 为第4层创建专属敌人，增强"终极挑战"感
2. 添加第4层专属奖励卡牌，提供最终层的独特收益
3. 更新游戏逻辑正确使用新内容

## 范围

### 包含
- 第4层专属敌人定义（2-3个）
- 第4层专属奖励卡牌（2-3张）
- 第4层专属精英（1个，备选）
- 游戏逻辑更新（层级敌人/卡池选择）

### 不包含
- 第4层地图结构修改
- 新Boss设计（黑莲老祖已有）
- 美术资源更新

## 验收标准
- [ ] 第4层战斗出现专属敌人（非第3层重复）
- [ ] 第4层奖励卡牌包含专属卡牌
- [ ] `tsc --noEmit` 通过
- [ ] `vite build` 通过
- [ ] `npm test` 全部通过
- [ ] 游戏可正常运行并体验第4层内容

## 开发计划

### 涉及文件
1. `src/data/cards.ts` - 添加第4层敌人和卡牌定义，导出新的常量
2. `src/game/store.ts` - 更新层级逻辑，使用第4层内容

### 实现步骤
1. 定义第4层敌人（2-3个）
2. 定义第4层奖励卡牌（2-3张）
3. 导出新的常量（ALL_ENEMIES_LAYER4, REWARD_CARDS_LAYER4）
4. 更新store.ts中的层级判断逻辑
5. 运行测试验证
6. 构建验证

### 敌人设计思路
- **天煞**：高血量+理智攻击组合，名字暗示"天降之煞"
- **心魔将**：精英型敌人，高伤害+自我增益
- **黑莲信徒**：黑莲老祖的追随者，高伤害+诅咒效果

### 卡牌设计思路
- **天命斩**：高费高伤，配合入魔效果更强
- **最终解脱**：消耗HP造成大量伤害
- **轮回护盾**：高护甲+自我回复
