# 需求文档：Python文字图片生成工具

## 背景

道诡修仙录（DBG游戏）目前使用运行时动态生成的 SVG 作为卡牌和敌人的占位图。这种方式：
- 图片样式简单，仅显示文字描述
- 不支持图标、符号等视觉元素
- 加载时需要实时生成，影响性能

## 目标

创建一个 Python 工具，批量生成高质量的卡牌/敌人图片，作为游戏占位图使用。

## 范围

### 包含
- Python 脚本生成 SVG/PNG 图片
- 支持从卡牌数据（cards.ts）读取配置
- 支持自定义样式（颜色、字体、图标）
- 支持简单几何图形和图标绘制
- 批量生成所有卡牌和敌人图片

### 不包含
- 真正的游戏美术资源（仅占位图）
- 复杂的 AI 图像生成
- 动画效果

## 验收标准

1. ✅ 工具能读取 `src/data/cards.ts` 中的卡牌定义
2. ✅ 为每张卡牌生成 160x220 的 SVG 图片
3. ✅ 为每个敌人生成 200x200 的 SVG 图片
4. ✅ 图片包含：卡牌名称、类型标识、简单图标
5. ✅ 输出到 `public/cards/` 和 `public/enemies/` 目录
6. ✅ 不同稀有度有视觉区分
7. ✅ 不同卡牌类型（攻击/技能/能力）有视觉区分

## 开发计划

### 涉及文件
- 新建：`tools/generate_images.py` - 主脚本

- 新建：`tools/styles.py` - 样式配置
- 新建：`tools/parse_cards.py` - 解析卡牌数据
- 新建：`public/cards/` - 输出目录
- 新建：`public/enemies/` - 输出目录
- 修改：`src/utils/placeholder.ts` - 优先使用静态图片
