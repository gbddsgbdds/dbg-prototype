# 需求文档：国际化支持（中英文切换）

## 背景
《道诡修仙录》目前仅支持中文界面。作为目标是成为成熟商品的游戏，需要支持多语言以面向全球玩家。英语是最重要的国际化语言，优先实现中英文切换。

## 目标
1. 实现中英文语言切换功能
2. 语言选择自动保存到localStorage
3. 用户可在设置界面切换语言

## 范围

### 包含
- 国际化框架搭建（使用react-i18next）
- 核心界面文本国际化
  - 开始界面
  - 角色选择界面
  - 战斗界面（状态栏、按钮）
  - 地图界面
  - 商店界面
  - 篝火界面
  - 胜利/失败界面
  - 设置界面
- 语言切换UI组件
- 语言偏好持久化

### 不包含
- 卡牌/敌人名称翻译（保持中文特色）
- 完整的帮助文档翻译
- E2E测试更新

## 验收标准
- [ ] 语言切换功能正常工作
- [ ] 核心界面文本正确显示中英文
- [ ] 语言偏好保存到localStorage
- [ ] `tsc --noEmit && vite build` 通过
- [ ] 游戏可正常运行

## 开发计划

### 修改文件
1. `package.json` - 添加i18next依赖
2. `src/i18n/` - 新建国际化配置目录
   - `index.ts` - i18n配置
   - `locales/zh.json` - 中文文本
   - `locales/en.json` - 英文文本
3. `src/App.tsx` - 初始化i18n
4. `src/components/Settings.tsx` - 添加语言切换
5. 各界面组件 - 替换硬编码文本

### 实现步骤
1. 安装react-i18next依赖
2. 创建i18n配置文件
3. 提取界面文本到语言文件
4. 修改组件使用i18n
5. 添加语言切换UI
6. 测试验证

## 翻译策略
- 保留游戏特色词汇（煞气、道心、入魔等）用拼音+英文解释
- 修仙术语保持中国特色
- UI通用词汇使用标准英文
