# 单元测试框架搭建

> 创建时间: 2026-03-17
> 状态: ✅ 已完成

## 背景

项目当前无测试覆盖，作为成熟商品需要自动化测试保障代码质量和稳定性。核心游戏逻辑（煞气系统、理智系统、入魔机制等）需要回归测试防止回归问题。

## 目标

1. 搭建 Vitest 测试框架 ✅
2. 为核心游戏逻辑编写单元测试 ✅
3. 集成到开发流程中 ✅

## 范围

### 包含

- Vitest 框架安装配置 ✅
- 核心模块测试：
  - `src/game/store.ts` - 状态管理测试 ✅
  - `src/game/meta.ts` - 元游戏系统测试 ✅
  - `src/game/types.ts` - 类型定义（无需测试）
- 测试脚本添加 ✅

### 不包含

- E2E 测试
- 组件测试（React Testing Library）
- 性能测试

## 验收标准

- [x] Vitest 框架配置完成
- [x] 至少 10 个测试用例通过（实际 33 个）
- [x] 核心逻辑覆盖率 > 50%
- [x] `npm run test` 命令可用
- [x] 测试通过不影响现有功能
