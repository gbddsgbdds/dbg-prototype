# 需求文档：增强音效系统

## 背景
当前音效系统使用简单的 Web Audio API 振荡器生成占位音效，音质较粗糙。可以通过更复杂的合成技术显著提升音效质量，为玩家提供更好的游戏体验。

## 目标
- 使用高级音频合成技术增强所有音效
- 保持零外部依赖（无需音频文件）
- 提升音效的真实感和游戏沉浸感

## 范围
### 包含
- 增强现有 9 种音效：attack, hit, heal, block, victory, defeat, card, click, madness
- 使用包络、滤波器、噪声、多层叠加等技术
- 添加音效变体（随机化避免重复感）

### 不包含
- 背景音乐（单独任务）
- 外部音频文件加载

## 验收标准
- [ ] 所有 9 种音效都有明显质量提升
- [ ] 音效有随机变体，避免单调
- [ ] 音效设置界面正常工作
- [ ] `tsc --noEmit && vite build` 通过
- [ ] 游戏运行无报错

## 开发计划

### 涉及文件
- `src/utils/soundManager.ts` - 主要修改

### 实现步骤
1. 添加音频合成辅助函数（包络、噪声、滤波器）
2. 重写 attack 音效（打击感）
3. 重写 hit 音效（受击反馈）
4. 重写 heal 音效（治愈感）
5. 重写 block 音效（金属质感）
6. 重写 victory 音效（胜利旋律）
7. 重写 defeat 音效（失败氛围）
8. 重写 card 音效（卡牌操作）
9. 重写 click 音效（UI交互）
10. 重写 madness 音效（入魔氛围）
11. 添加随机变体系统
12. 测试验证
