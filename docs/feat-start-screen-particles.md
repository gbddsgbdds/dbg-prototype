# 需求文档：开始界面背景粒子动画

## 背景
当前开始界面背景是静态渐变，缺少动态感。作为一款修仙题材的DBG游戏，开始界面应该有"灵气流动"的氛围感。

## 目标
为开始界面添加动态粒子效果，提升视觉品质和沉浸感。

## 范围
- 在开始界面添加浮动粒子动画
- 粒子样式符合修仙主题（光点、光晕）
- 性能友好，不影响加载和交互

## 不做
- 不修改其他界面
- 不添加复杂的Canvas/WebGL粒子系统（用CSS实现）

## 验收标准
1. 开始界面有浮动的光点粒子动画
2. 粒子颜色与整体风格协调
3. 不影响按钮点击和界面加载
4. 通过 tsc --noEmit && vite build 验证

## 开发计划

### 涉及文件
- `src/App.css` - 添加粒子动画CSS
- `src/App.tsx` - 添加粒子DOM元素

### 实现步骤
1. 在App.css中定义粒子动画关键帧
2. 在App.tsx开始界面添加粒子容器
3. 验证动画效果和性能
