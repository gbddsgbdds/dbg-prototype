# 更新日志

所有重要的更改都将记录在此文件中。

## [0.6.4] - 2026-03-17

### 新增
- **角色解锁系统**
  - 新增角色解锁机制，部分角色需要达成条件才能解锁
  - 张道陵：使用心素通关一次后解锁
  - 骰子：在入魔状态下击败Boss后解锁
  - 角色选择界面显示锁定状态和解锁条件提示
  - 角色解锁弹窗动画效果
- **胜利界面优化**
  - 通关时自动检测并触发角色解锁

### 改进
- 游戏版本升级到 v0.6.4
- 角色选择界面视觉优化，锁定角色显示问号和锁定遮罩

## [0.6.3] - 2026-03-17

### 新增
- **国际化支持（i18n）**
  - 集成 i18next + react-i18next 国际化框架
  - 支持中英文语言切换
  - 设置界面添加语言选择按钮
  - 语言偏好自动保存到 localStorage
  - 核心界面文本国际化（开始界面、战斗界面、地图、商店、胜利/失败界面等）

### 改进
- 游戏版本升级到 v0.6.3
- 包体积略微增加（+56KB gzip后约+18KB），引入i18n依赖

## [0.6.2] - 2026-03-17

### 新增
- **第4层专属内容扩展**
  - 3 个第4层专属敌人：天煞（高伤害+理智攻击）、心魔将（高血量+力量成长）、黑莲信徒（诅咒型）
  - 1 个第4层专属精英：黑莲护法（高血量+理智攻击+力量成长）
  - 3 张第4层专属卡牌：天命斩（入魔加成）、最终解脱（HP换伤害）、轮回护盾（高护甲+回复）
  - 新增 'madness' 条件类型，支持入魔状态条件判断
  - 新增 damageBonus 属性，支持条件满足时固定伤害加成

### 改进
- 游戏版本升级到 v0.6.2
- 第4层战斗使用专属敌人，不再复用第3层内容
- 第4层奖励卡池使用专属卡牌
